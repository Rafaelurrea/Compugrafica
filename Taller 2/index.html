Punto 2

Titulo h1

Función: Encabezado principal, generalmente utilizado para el título más importante de la página como el nombre del sitio o artículo.

SEO: Los motores de búsqueda le dan mayor relevancia, así que solo debería haber uno por página.

Estilo predeterminado: Texto grande y en negrita.


Titulo h2

Función: Subtítulo directo del h1. Se usa para dividir el contenido en secciones principales.

SEO:También importante, aunque con menor peso que el h1.

Estilo predeterminado: Más pequeño que el h1.


Titulo h3

Función: Encabezado de subsecciones dentro de un h2. Por ejemplo, para dividir temas específicos de una sección.

SEO: Útil para estructurar el contenido, ayudando a que los motores de búsqueda entiendan la relación jerárquica.

Estilo predeterminado: Más pequeño que el h2.


Titulo h4

Función: Títulos dentro de subsecciones más pequeñas, como categorías o temas muy específicos.

SEO: Aporta orden, aunque tiene menor impacto en los motores de búsqueda.

Estilo predeterminado: Más pequeño que el h3.


Titulo h5

Función: Se usa para detalles aún más específicos, como subtítulos dentro de una sección h4.

SEO: No tiene mucho peso, pero ayuda a la organización visual.

Estilo predeterminado: Más pequeño que el h4.


Titulo h6

Función: El nivel más bajo, usado para encabezados muy pequeños o información secundaria.

SEO: Prácticamente sin impacto directo en los motores de búsqueda, más útil para diseño visual.

Estilo predeterminado: El tamaño de texto más pequeño dentro de los encabezados.


IMG

Muestra imágenes en la página web.
Es una etiqueta vacía (no tiene etiqueta de cierre).
Requiere al menos el atributo src (ruta de la imagen).

Atributos comunes

SRC: Especifica la ubicación de la imagen.
ALT: Texto alternativo que aparece si la imagen no se carga (también importante para accesibilidad).
WIDTH y HEIGTH: Controlan el tamaño de la imagen.


VIDEO

Atributos comunes

src: Ruta del archivo de video.
controls: Añade controles de reproducción (play, pausa, volumen).
autoplay: Hace que el video se reproduzca automáticamente.
loop: Reproduce el video en bucle

Atributos comunes

Se usa para organizar datos en filas y columnas. Dentro de una tabla, se utilizan etiquetas adicionales:
tr (table row): define una fila.
th (table header): celda de encabezado.
td (table data): celda de datos.


Atributos comunes

Atributos comunes:
cols: Número de columnas visibles.
rows: Número de filas visibles.
border: Añade bordes a la tabla.
cellpadding: Define el espacio dentro de cada celda.
cellspacing: Controla el espacio entre las celdas.
